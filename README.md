# PDF‑to‑Excel Report Generator

A lightweight utility to convert tabular data from PDF files into a structured Excel report.

## Description

This tool extracts tabular data from a PDF file (located in the same directory as the script) and generates an Excel report. Ideal for automating routine document processing tasks.

## How to Use

1. Place your source PDF file in the project folder (same directory as `main.py`).
2. Run the script:
   ```bash
   python main.py
